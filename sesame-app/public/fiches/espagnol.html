Espagnol
LV2

Notions clés
⚔️
SER vs ESTAR (piège n°1)
  • SER = caractéristiques permanentes : identité, nationalité, profession, origine, heure
  • Soy estudiante · Son las 3
  • ESTAR = états temporaires, localisation, résultat d'un changement
  • Estoy cansado · El hospital está aquí
  • Piège : estar muerto=état (mort) · ser malo=caractère mauvais · estar malo=être malade
🏠
HAY vs ESTAR (piège n°2)
  • HAY = existence indéfinie : Hay un hospital (il y a un hôpital)
  • ESTAR = localisation d'un élément défini : El hospital está en la calle Mayor
  • Règle : article indéfini / pas d'article → hay
  • Règle : article défini → está / están
🔀
POR vs PARA (piège n°3)
  • POR = cause, moyen, durée, échange, agent passif
  • Lo hago por ti · por avión · por 3 horas
  • PARA = but, destination, opinion, délai
  • Voy para Madrid · para mí · para el lunes
  • Mnémo : POR = raison passée/cause · PARA = objectif futur/but
🔀
Les 3 hypothèses avec "si"
  • Réel : si + présent → futur
  • Si llueve, no iré
  • Potentiel : si + imparfait subj. → conditionnel
  • Si tuviera dinero, viajaría
  • Irréel passé : si + plus-que-parfait subj. → conditionnel passé
  • Si hubieras venido, lo habrías visto
🌍
L'espace andin
  • 7 pays traversés par les Andes : Venezuela, Colombie, Équateur, Pérou, Bolivie, Chili, Argentine
  • Héritage inca : Machu Picchu, quechua, aymara
  • Défis : pauvreté, inégalités, déforestation, migration urbaine
  • Diversité ethnique : métis, indigènes, descendants coloniaux
📝
Subjonctif espagnol
  • Souhaits / volonté : quiero que, ojalá + subj.
  • Doute / opinion négative : no creo que, dudo que
  • Nécessité : es importante que, es necesario que
  • Sentiment : me alegra que, me sorprende que
🔗
Articulateurs du discours
  • Chronologiques : en primer lugar, luego, después, finalmente
  • Opposition : sin embargo, no obstante, aunque
  • Cause : porque, ya que, puesto que
  • Conséquence : por eso, por lo tanto, así que
  • Addition : además, también, incluso
🚫
Négations espagnoles
  • no...nunca (jamais) · no...nadie (personne) · no...nada (rien)
  • ya no (ne plus) · tampoco (non plus)
  • Si le mot négatif est avant le verbe, pas besoin de "no"
  • Nunca como tarde = No como nunca tarde — les deux sont corrects !
Conjugaison — 4 blocs structurés
⚡ Présent irréguliers clés
SER : soy / eres / es / somos / sois / son ESTAR: estoy / estás / está / estamos / estáis / están IR : voy / vas / va / vamos / vais / van TENER: tengo / tienes / tiene / tenemos / tenéis / tienen HACER: hago / haces / hace / hacemos / hacéis / hacen
Astuce : la 1re pers. sg. est souvent irrégulière (tengo, hago, voy…). Les autres suivent généralement un schéma régulier.
⏳ Les 3 passés
IMPERFECTO → état / habitude passée hablar : hablaba / hablabas / hablaba vivir : vivía / vivías / vivía INDEFINIDO → action délimitée dans le temps hablar : hablé / hablaste / habló vivir : viví / viviste / vivió PLUSCUAMPERFECTO → avant une autre action passée había comido / habías comido / había comido
Règle : Indefinido = action précise, ponctuelle. Imperfecto = description, répétition, contexte. Plus-que-parfait = antériorité dans le passé.
🔮 Futur & conditionnel
FUTURO SIMPLE hablaré / hablarás / hablará hablaremos / hablaréis / hablarán CONDICIONAL hablaría / hablarías / hablaría hablaríamos / hablaríais / hablarían FUTURO PRÓXIMO (ir + a + inf.) voy a hablar / vas a comer / va a vivir
Piège : jamais de conditionnel après "si" hypothétique ! Si + imparfait subj. (tuviera) → conditionnel dans la principale (viajaría).
🔁 Verbes pronominaux & gustar
RÉFLÉCHIS : lavarse (se laver) me lavo / te lavas / se lava RÉCIPROQUES : llamarse me llamo / te llamas / se llama GUSTAR (verbe à l'envers !) me gusta / te gusta / le gusta (sing.) me gustan / te gustan / le gustan (plur.) DOLER (avoir mal) me duele la cabeza / me duelen los pies
Clé : avec gustar / doler, c'est la chose aimée/douloureuse qui est sujet — accord avec elle, pas avec la personne.
Pièges classiques — erreurs fréquentes
❌ Piège
"Estoy estudiante"
✓ Correct
"Soy estudiante" — la profession est une caractéristique permanente → SER, pas ESTAR.
❌ Piège
"Hay el hospital aquí"
✓ Correct
"El hospital está aquí" — élément défini (article défini) + localisation → ESTAR, pas HAY.
❌ Piège
"Si vendría…"
✓ Correct
Jamais de conditionnel après "si" hypothétique ! → "Si viniera…" (imparfait du subjonctif).
❌ Piège
"dirección" = uniquement direction
✓ Correct
Faux-ami partiel : "dirección" = direction ET adresse. Toujours vérifier le contexte.
❌ Piège
"No como nunca tarde" est incorrect
✓ Correct
Les deux formes sont correctes ! "No como nunca tarde" (double négation) = "Nunca como tarde" — la double négation est grammaticale en espagnol.
❌ Piège
"Para mí" = "Por mí" (équivalents)
✓ Correct
"Para mí, tiene razón" = selon moi (opinion). "Por mí, puede ir" = en ce qui me concerne / à cause de moi (cause/permission). Sens très différents !
Quiz éclair — clique pour voir la réponse
SER ou ESTAR : "La soupe est froide"
Está fría — état temporaire (la soupe peut se réchauffer) → ESTAR. Si c'était une caractéristique permanente, on utiliserait SER.

HAY ou ESTÁ : "Il y a une pharmacie près d'ici"
Hay una farmacia cerca de aquí — existence indéfinie (une pharmacie, pas "la pharmacie") → HAY avec article indéfini.

POR ou PARA : "Je t'appelle pour te remercier"
Te llamo para agradecerte — but / objectif → PARA. (POR exprimerait la cause : "Te llamo por el problema" = je t'appelle à cause du problème.)
