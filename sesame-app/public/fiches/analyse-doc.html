Séquence 3 — Fiche mémo
Analyse
documentaire
20 questions · 2h 3 parties progressives Dossier documentaire fourni
⚠ Règle d'or — Répondre UNIQUEMENT avec les documents. Jamais avec ses connaissances personnelles.
Structure de l'épreuve
📍
Partie 1 — Repérage (6 pts)
  • 6 questions / 4 propositions / 1 seule bonne réponse
  • Objectif : REPÉRER une info dans UN seul document
  • Lire tableaux, courbes, histogrammes, diagrammes circulaires
  • Compétence : localiser rapidement une donnée précise
🔍
Partie 2 — Analyse (6 pts)
  • 6 questions / 6 propositions / PLUSIEURS bonnes réponses
  • Objectif : ANALYSER et RELIER des infos de PLUSIEURS documents
  • ⚠ 1 mauvaise sélectionnée = 0 point à TOUTE la question
  • Stratégie : brouillon obligatoire avant de cocher
🏗️
Partie 3 — Synthèse (8 pts)
  • 8 questions / 4 propositions / 1 seule bonne réponse
  • Objectif : CONSTRUIRE UN RAISONNEMENT sur l'intro, le plan, le développement, la conclusion
  • Plan dialectique : thèse / antithèse / synthèse
  • Plan analytique : 3 axes descriptifs
📊
Outils statistiques requis
  • % répartition = valeur / total × 100
  • Taux variation = (Vf − Vi) / Vi × 100
  • Coeff. multiplicateur = Vf / Vi
  • Indice simple = (valeur / valeur_base) × 100
  • Quantiles : médiane (50 %), quartiles (25 %/75 %), déciles
⏱️
Stratégie temporelle
  • 2h au total pour ~20 questions
  • Lire TOUT le dossier en premier (~45–60 min)
  • Ne jamais sauter la lecture complète
  • Temps restant : répondre + relecture
  • Partie 2 (analyse) : faire brouillon avant de cocher
🚫
Règle fondamentale
  • JAMAIS utiliser ses connaissances personnelles
  • TOUT doit venir des documents fournis
  • Si ça n'est pas dans le dossier → ce n'est pas la bonne réponse
  • L'épreuve teste la méthode, pas la culture générale
Méthode — Lecture des documents
📋 Lire un tableau
1. Lire le TITRE 2. Lire les EN-TÊTES (lignes + colonnes) 3. Identifier l'UNITÉ (€, %, milliers…) 4. Vérifier la SOURCE 5. Repérer la BASE (total = 100% pour répartition) → Puis seulement : calculer / comparer
Piège : confondre lignes et colonnes lors du croisement. Repérer le total général avant tout calcul de répartition.
📈 Lire une courbe / graphique
1. Lire le TITRE 2. Identifier les AXES (x = temps? catégorie?) 3. Lire la LÉGENDE (plusieurs séries?) 4. Identifier l'UNITÉ de chaque axe 5. Dégager la TENDANCE (hausse / baisse / stable) → Si nécessaire : calculer la variation
Méthode variation : taux = (Vf − Vi) / Vi × 100. Toujours vérifier l'unité de l'axe Y avant de lire une valeur.
🔗 Croiser des documents (Partie 2)
Pour chaque proposition : → Dans QUEL document se trouve l'info ? → Existe-t-il une PREUVE textuelle ou statistique dans ce doc ? → OUI → cocher / NON → ne pas cocher Brouillon : numéroter les docs (D1, D2…) noter les liens entre eux
Rappel : 1 erreur = 0 point sur la question entière. Mieux vaut cocher 3 certaines que 5 approximatives.
🏗️ Structure d'une synthèse correcte
INTRODUCTION Accroche → chiffre / fait frappant du dossier Définition → termes du sujet Problématique → question centrale Annonce → plan en 2 ou 3 parties DÉVELOPPEMENT Thèse → argument + illustration docs Antithèse → nuance / contre-argument Synthèse → dépassement (si dialectique) CONCLUSION Bilan → réponse à la problématique Ouverture → perspective / question plus large
Choix du plan : dialectique si sujet débattu (pour/contre). Analytique si sujet à décrire/analyser sans opposition.
Pièges classiques
❌ Erreur
Utiliser une connaissance perso non présente dans le dossier
✅ Bonne pratique
Si c'est pas dans les docs, c'est faux — point final
❌ Erreur
En Partie 2 : cocher "presque toutes" les propositions par précaution
✅ Bonne pratique
1 erreur = 0 point total. Cocher uniquement ce qui est certain avec preuve documentaire
❌ Erreur
Confondre taux de variation et coefficient multiplicateur
✅ Bonne pratique
CM = 1,25 ≠ taux = 25 %. Relation : CM = 1 + taux/100. Ex : hausse de 50 % → CM = 1,5
❌ Erreur
Lire une répartition comme une évolution dans le temps
✅ Bonne pratique
Répartition = parts du gâteau à UN instant T, pas une comparaison temporelle
❌ Erreur
Plan dialectique pour un sujet non polémique / purement descriptif
✅ Bonne pratique
Dialectique = pour les débats (pour/contre). Analytique = pour descriptions et analyses neutres
❌ Erreur
Commencer à répondre sans avoir lu tout le dossier
✅ Bonne pratique
Des documents plus loin peuvent contredire ou compléter les premiers. Lire TOUT d'abord
Quiz éclair — Cliquer pour révéler la réponse
Un document montre que les ventes sont passées de 80 à 120. Quel est le CM ?
→ Cliquer pour voir la réponse
CM = 120 / 80 = 1,5 — ce qui correspond à une hausse de 50 %
Taux de variation = (120 − 80) / 80 × 100 = 50 %
En Partie 2, tu n'es sûr(e) que de 3 propositions sur 6. Tu fais quoi ?
→ Cliquer pour voir la réponse
Ne cocher que les 3 certaines. 1 erreur = 0 point total sur toute la question. La prudence est rentable.
Quelle est la structure d'une introduction de synthèse ?
→ Cliquer pour voir la réponse
Accroche (chiffre frappant du dossier) → Définition des termes du sujet → Problématique → Annonce du plan
Différence entre plan dialectique et plan analytique ?
→ Cliquer pour voir la réponse
Dialectique = thèse / antithèse / synthèse → pour les sujets débattus (pour/contre, avantages/inconvénients)
Analytique = 3 axes descriptifs → pour les sujets à analyser sans opposition frontale
Un indice passe de 100 à 135. Quelle est l'évolution en % ?
→ Cliquer pour voir la réponse
+35 % — L'indice 135 en base 100 signifie une hausse de 35 points, soit 35 % par rapport à la valeur de base.
