Notions clés
🔀
Les 3 hypothèses
  • Réel : si + présent → futur si tu viens, je serai là
  • Potentiel : si + imparfait → conditionnel si tu venais, je serais là
  • Irréel passé : si + plus-que-parfait → conditionnel passé si tu étais venu, j'aurais été là
  • ⚠ Jamais de conditionnel après "si" hypothétique
🔧
Accord du participe passé
  • Avec ÊTRE : accord avec le sujet
  • Avec AVOIR : accord avec le COD si placé avant
  • Pronominaux : accord avec le sujet sauf si COD après
  • Ex: elle s'est lavé les mains → lavé invariable (COD après)
📝
5 types d'exercices officiels
  • Analogies verbales : X est à Y ce que Z est à ___
  • Maladresses d'expression : trouver la phrase sans faute
  • Unités textuelles : texte à trous cohérent
  • Vocabulaire & orthographe : bon mot + bonne graphie
  • Synonymes / Antonymes / Homonymes / Paronymes
🎭
Subjonctif vs Indicatif
  • Subjonctif après : vouloir que, falloir que, pour que, bien que, à moins que, opinion négative
  • Indicatif après : penser que (affirmatif), savoir que, croire que (affirmatif)
  • Formule clé : doute / sentiment / volonté → subjonctif
🔁
Pronoms
  • y / en : "y" remplace à+lieu ou à+chose, "en" remplace de+chose
  • Doubles pronoms : me le, te la, lui en, y en
  • Relatifs complexes : auquel, duquel, parmi lesquels
📖
Culture littéraire
  • Poésie : Moyen-Âge → XVIIe
  • Littérature d'idée & presse : XVIe → XXIe
  • Roman & récit : XVIIIe → XXIe
  • Théâtre : XVIIe → XXIe
🔗
Articulateurs logiques
  • Opposition : mais, cependant, pourtant, néanmoins, or
  • Cause : car, parce que, puisque, étant donné que
  • But : pour que, afin que (+subj), pour (+inf)
  • Concession : bien que, quoique, même si
💡
Valeurs de "on"
  • NOUS (registre familier) : "On part demain"
  • ILS / quelques personnes : "On dit que…"
  • QUELQU'UN (indéfini) : "On frappe à la porte"
  • ⚠ Piège fréquent en QCM : identifier la bonne valeur
Conjugaison clé
Présent — irréguliers essentiels
être → suis/es/est/sommes/êtes/sont avoir → ai/as/a/avons/avez/ont aller → vais/vas/va/allons/allez/vont faire → fais/fais/fait/faisons/faites/font
Formes à retenir par cœur — piège : faisons (pas "faisons" avec â)
Passé composé vs Imparfait
PC = action ponctuelle / délimitée → "j'ai mangé" (fini, daté) IMP = état / habitude / description → "je mangeais" (durée, contexte)
Test : "à ce moment-là" → imparfait / "soudain" → passé composé
Discours rapporté au passé
présent → imparfait futur → conditionnel passé comp → plus-que-parfait "demain" → "le lendemain" "hier" → "la veille" "aujourd'hui" → "ce jour-là"
Règle de la concordance des temps au discours indirect
Gérondif vs Participe présent
Gérondif = EN + V-ant → simultanéité, même sujet → "en chantant, il travaille" Participe présent = V-ant seul → qui fait l'action → "chantant faux, il fut hué"
Le gérondif exige le même sujet que le verbe principal
Pièges classiques
❌ Erreur
"Elle s'est lavé les mains" — accorder lavé ?
✅ Correct
"lavé" est invariable : le COD "les mains" est après le verbe
❌ Erreur
Confondre "conjoncture" et "conjecture"
✅ Correct
conjoncture = situation économique · conjecture = hypothèse
❌ Erreur
"si j'aurais su, je n'aurais pas venu"
✅ Correct
Jamais de conditionnel après "si" hypothétique → si j'avais su
❌ Erreur
"bien que" + indicatif
✅ Correct
"bien que" exige toujours le subjonctif → "bien que tu sois"
❌ Erreur
Confondre "a" (avoir) et "à" (préposition)
✅ Correct
Test : remplacer par "avait" — si possible → "a" conjugué, sinon → "à"
❌ Erreur
Analogie verbale : chercher le mauvais rapport
✅ Correct
Identifier d'abord la relation (tout/partie, cause/effet, synonyme…) puis appliquer
Quiz éclair — cliquer pour révéler
1. Quel temps utiliser après "bien que" ?
Le subjonctif — toujours.
Ex : "bien que tu sois là" / "bien qu'il fasse froid"
2. Accordez : "Les robes qu'elle a portées" — pourquoi l'accord ?
portées s'accorde car le COD "que" (= les robes, féminin pluriel) est placé avant le verbe avoir.
Règle : avec avoir → accord avec le COD antéposé
3. Donnez 3 synonymes de "acrimonieux".
Acerbe, mordant, virulent (aussi : caustique, cinglant, âpre)
Acrimonieux = plein d'aigreur et d'hostilité dans les propos
4. "Partir" est à "arriver" ce que "vendre" est à ___ ?
Acheter
Relation d'antonymes — partir ↔ arriver / vendre ↔ acheter
5. Quelle valeur de "on" dans : "On frappe à la porte" ?
Valeur indéfinie : quelqu'un (on ne sait pas qui).
≠ valeur "nous" (familier) · ≠ valeur "ils/les gens" (générale)
