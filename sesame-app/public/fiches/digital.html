Notions clés
📡
Flux RSS & information
  • RSS = Really Simple Syndication — flux d'info automatisé
  • Recevoir l'info sans aller la chercher = agrégation
  • Outil : agrégateur de flux (ex: Feedly)
  • ≠ concaténation (assemblage de texte), diffusion (1→tous)
🔍
Moteurs de recherche
  • Efficacité = bien choisir ses mots-clés
  • Questions longues → moins précis
  • 1 seul mot-clé → trop large
  • Opérateurs : guillemets pour exact, "-" pour exclure
🏷️
Métadonnées
  • Données sur les données : auteur, titre, date, description
  • PAS une métadonnée : les valeurs d'un formulaire
  • Utilisées par les moteurs de recherche (SEO)
  • Exemple : EXIF dans une photo (GPS, appareil, date)
🗜️
Compression de fichiers
  • Logiciels : 7ZIP, WinZIP, WinRAR
  • Permet : réduire taille, archiver plusieurs fichiers, protéger par mot de passe
  • Ne permet PAS : diviser en parties de tailles identiques (≠ split)
  • Archive = conteneur compressé
📊
Tableur — références
  • Référence relative : A1 → se déplace à la recopie
  • Référence absolue : $A$1 → fixe (2 symboles $)
  • Référence mixte : $A1 ou A$1
  • Formule TVA correcte : =B2*$E$2 (fixer E2, pas B2)
📧
Communication numérique
  • Newsletter → envoyée via liste de diffusion
  • Mode collaboratif = plusieurs utilisateurs sur 1 document partagé
  • Indicateur de visibilité = vues, partages, visites
  • PAS un indicateur : nombre de médias publiés
📁
Formats de fichiers
  • .docx = Word (texte)
  • .xlsx = Excel (tableur)
  • .xlsm = Excel avec macros → risque virus
  • .zip / .arj = archives compressées ≠ texte
  • PDF créé par : Acrobat, Word, Excel — pas Visual Studio
🔒
Sécurité numérique
  • Double authentification : SMS, email, empreinte digitale
  • PAS une 2e auth : ressaisir son nom d'utilisateur
  • Usurpation d'identité → utiliser un pseudonyme
  • Fichier dangereux par email : .xlsm (macros)
  • Data centers ≈ impact GES du transport aérien mondial
Algorithmes — 4 types à maîtriser
🔁 Boucle Pour (for)
N = 1 pour i de 1 à 10 faire N = N + 2 fin pour afficher(N) → 21
Méthode : 10 itérations × (+2) = +20. N démarre à 1 → 1+20 = 21
Attention : la boucle s'exécute N fois, pas N-1.
🔄 Répéter Jusqu'à
A=2, B=17 répéter B = B - A → 15, 13, 11, 9, 7, 5, 3, 1 jusqu'à B >= A → stop quand 1 >= 2 ? NON → stop quand 3 >= 2 ? OUI → afficher 1
Piège : la condition est évaluée après l'exécution. Simuler pas à pas : 17→15→13→11→9→7→5→3→1. On s'arrête quand B≥A, donc quand B revient à 1 (après avoir été soustrait une dernière fois de 3).
📋 Listes (tableaux)
N=5, Valeurs[i] = i*2+1 → [3, 5, 7, 9, 11] puis Valeurs[i] += Valeurs[i-1] i=2: 5+3=8 i=3: 7+8=15 i=4: 9+15=24 i=5: 11+24=35
Méthode : 2 passes. 1re : remplir le tableau. 2e : accumuler. Chaque case devient la somme avec la précédente.
✅ Booléen & Répéter
i=3, k=15, N=1, Continu=VRAI répéter Continu = (k > i) → T, T, T, T, F k = k - i → 12, 9, 6, 3, 0 N = N + 1 → 2, 3, 4, 5, 6 jusqu'à Continu → stop quand Continu=VRAI (après itération)
Résultat : N=6 et k=0. Piège : "jusqu'à Continu" = s'arrête quand Continu est VRAI, pas faux. La condition est testée après que k et N ont été mis à jour.
Pièges classiques — confusion fréquente
❌ Piège
Agrégation = concaténation
✓ Correct
Agrégation = collecter/regrouper des flux. Concaténation = assembler des chaînes de texte.
❌ Piège
Référence absolue = 1 seul $
✓ Correct
Référence absolue = $A$1 (2 symboles $). Mixte = 1 seul $.
❌ Piège
ZIP peut diviser un fichier en parties identiques
✓ Correct
Non — c'est le "split" qui fait ça. ZIP compresse et archive, il ne divise pas.
❌ Piège
Nombre de médias publiés = indicateur de visibilité
✓ Correct
Non — publier beaucoup ≠ être vu. Les vrais indicateurs sont vues, partages, visites.
❌ Piège
Visual Studio crée des PDF
✓ Correct
Non — VS est un IDE pour coder. Les PDF sont créés par Word, Excel, Acrobat Reader.

Quelle est la formule correcte pour calculer le prix TTC en colonne C en fixant le taux de TVA en E2 ?
=B2*$E$2 — B2 est relatif (se déplace à la recopie), $E$2 est absolu (fixe le taux de TVA).

Quel format de fichier joint dans un mail représente un risque de virus particulier ?
.xlsm — fichier Excel avec macros. Les macros peuvent contenir du code malveillant qui s'exécute à l'ouverture.

Qu'affiche l'algo : N=1, pour i de 1 à 10, N=N+2 ?
21 — N part de 1, on ajoute 2 dix fois : 1 + (10×2) = 21.

Quel moyen n'est PAS utilisé dans la double authentification ?
Ressaisir son nom d'utilisateur — Ce n'est pas une vérification d'identité supplémentaire, c'est déjà la 1re étape.

L'impact des data centers sur les GES est comparable à quel moyen de transport mondial ?
Transport aérien — Les data centers consomment énormément d'électricité et représentent ~2-3% des émissions mondiales, similaire à l'aviation.
